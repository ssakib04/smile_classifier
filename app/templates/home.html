Project Overview

Framework & Architecture

This application is built using FastAPI for the web API backend, SQLAlchemy ORM connected to PostgreSQL for logging, and scikit-learn for model training and inference.

Model & Preprocessing

  • Model: Random Forest Classifier (Scikit-Learn). It handles small-to-medium datasets robustly without overfitting quickly.
  • Feature Pipeline: Images uploaded during training/inference are resized to 64×64 pixels, converted to RGB, normalized into pixel vectors ([0, 1] scale), and passed to the classifier.
  • Image Normalization: All uploads are converted to standard .jpg format. Transparency (RGBA/LA) is composited onto a white background before saving.
  • Inference Flow: Single image uploads are stored in the file system and logged alongside prediction results and confidence metrics in PostgreSQL.
