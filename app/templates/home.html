Project Overview

Framework & Architecture Choice

This application is built using FastAPI for the web API backend, SQLAlchemy ORM connected to PostgreSQL for logging, and scikit-learn for model training and inference.

Model & Preprocessing Explanation

  • Model: Random Forest Classifier (Scikit-Learn)[cite: 1]. It handles small-to-medium datasets robustly without overfitting quickly.
  • Feature Pipeline: Images uploaded during training/inference are resized to 64x64 pixels, converted to RGB, normalized into pixel vectors ($[0, 1]$ scale), and passed to the classifier.
  • Inference Flow: Single image uploads are automatically converted to standard .jpg, stored in the file system, and logged alongside prediction results and confidence metrics in PostgreSQL[cite: 1].
